## To‑Do List (React + TypeScript + Vite)

Современное приложение «Список дел», демонстрирующее продвинутые практики фронтенда: чистые типы, reducer + Context, локальная персистентность, доступность и аккуратный UI.

### Стек
- React 18, TypeScript
- Vite 5
- Без сторонних стейт-менеджеров — `useReducer` + Context

### Возможности
- Добавление задач с названием, сроком, тегами, приоритетом
- Редактирование, удаление, переключение «выполнено»
- Фильтры по статусу, тегу и приоритету
- Поиск по названию/заметкам/тегам
- Сортировка по дате создания/обновления, названию, сроку, приоритету
- Локальное сохранение в `localStorage`
- Клавиатурная доступность, aria-атрибуты
- Адаптивный интерфейс


### Архитектура
- `src/types.ts` — доменные типы (Todo, фильтры, сортировка)
- `src/utils/todoUtils.ts` — pure-функции: создание/обновление, фильтры и сортировка
- `src/state/TodoContext.tsx` — reducer + Context, персистентность в `localStorage`
- `src/components/TodoApp.tsx` — композиция UI: форма, тулбар, список
- `src/styles.css` — современная тема с авто-светлой/тёмной схемой

### Качество кода
- Строгая типизация, отсутствие `any`
- Чистые функции, разделение домена и UI
- Минимум побочных эффектов, удобный для тестирования код в `utils`
- Доступность: aria-атрибуты, семантические роли, управление с клавиатуры

### Идеи для развития
- Е2Е и unit‑тесты (Vitest + Testing Library)
- Экспорт/импорт задач в JSON
- Мультиборды/проекты, drag‑and‑drop порядка задач
